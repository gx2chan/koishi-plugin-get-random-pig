# koishi-plugin-get-random-pig

[![npm](https://img.shields.io/npm/v/koishi-plugin-get-random-pig?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-get-random-pig)

依靠着<https://pighub.top>资源的随机猪猪群友插件哦

# 插件使用说明

- 接上[PigHub](https://pighub.top)的随机猪猪群友图片插件

- 在群里有人发送你设置的猪猪触发词时返回一个随机猪猪！！！ <img src="https://ywh555hhh.github.io/EmoGdream/stickers/KB/kb_wake_up.webp" alt="kb_wake_up.webp" width="24" align="absmiddle">

- 找猪功能已部署，使用/找猪 （猪猪名）来尝试找到你想要的猪猪吧 <img src="https://ywh555hhh.github.io/EmoGdream/stickers/ryo/ryo_thinking.png" alt="ryo_thinking.png" width="24" align="absmiddle">

使用示例： `/找猪 猪睡觉`、`/猪`
